AskUI Desktop is the local host for authoring and running AI-driven UI tests.
It runs on Windows and macOS.

<Steps>

1. **Download the installer** for your platform from your AskUI account.

2. **Run the installer** and follow the guided setup.

   <Tabs>
     <TabItem label="Windows">
       Run the `.exe` and accept the prompts. AskUI Desktop installs with an
       embedded Python runtime — no separate Python install is required.
     </TabItem>
     <TabItem label="macOS">
       Open the `.dmg` and drag AskUI Desktop to Applications. On first launch,
       grant **Accessibility** and **Screen Recording** permissions under
       **System Settings → Privacy & Security** so the agent can read the
       screen and control input.
     </TabItem>
   </Tabs>

3. **Sign in** with your AskUI account — see [Sign in & onboarding](/get-started/onboarding/).

4. **Open or create a Project** and run [your first test](/get-started/first-run/).

</Steps>
