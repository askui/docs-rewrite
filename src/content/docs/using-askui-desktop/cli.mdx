The CLI runs the exact Projects you build in AskUI Desktop. The same `tests/`,
`prompts/`, `plans/`, and `procedures/` drive both surfaces — author in the
app, run anywhere with the CLI.

```bash
# scaffold — final flags TBD
askui run --project ./my-project --plan smoke
```
